🎲 Python Dice Simulator

A simple command-line Dice Simulator built using Python.
The program simulates rolling a dice and generates a random number between 1 and 6.

---

🚀 Features

- Simulates a dice roll
- Generates random numbers between 1 and 6
- Menu-based user interaction
- Continuous play option
- Input validation

---

🛠 Technologies Used

- Python 3
- Random module

▶️ How to Run

1. Clone the repository

git clone https://github.com/saba-abbasi-381/python-dice-simulator.git

2. Navigate to the project folder

cd python-dice-simulator

3. Run the program

python dice_simulator.py
